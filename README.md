## AI MCQ Generator Project

This project is designed to generate multiple-choice questions (MCQs) using artificial intelligence. The AI model is trained on a large dataset of questions and answers, allowing it to create new questions based on the input provided.

### Features
- Generate MCQs based on a given topic or subject.
- Customize the number of questions and answer options.
- Provide explanations for the correct answers.
- Support for various subjects and difficulty levels.

### Technologies Used
streamlit
google-genai
python-dotenv
reportlab
PyPDF2
pdfplumber

### Streamlit
Streamlit is an open-source app framework used for creating and sharing data applications. It allows developers to build interactive web applications with ease, using simple Python scripts.

### Google GenAI
Google GenAI is a powerful language model developed by Google that can generate human-like text based on the input it receives. It is used in this project to create MCQs by understanding the context and generating relevant questions and answers.
